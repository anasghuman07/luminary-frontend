# LUMIN.ARY

Lumin.ary is a modern, elegant, and fully responsive landing page designed to showcase a premium productivity and focus-enhancement platform. Built with pure HTML, CSS, and JavaScript, it features smooth navigation, beautiful UI components, interactive modals, pricing sections, and a polished user experience.

## ✨ Features

* Modern luxury-inspired UI design
* Fully responsive layout
* Smooth single-page navigation
* Interactive Login & Signup modals
* Beautiful hero section with call-to-action
* Pricing plans section
* About & team showcase
* Contact forms and information panels
* Toast notification system
* Smooth animations and transitions
* Mobile-friendly design

## 🛠 Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Google Fonts
* Cormorant Garamond
* DM Sans
